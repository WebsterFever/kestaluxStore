# Kestalux Cosmetics — Vanilla JS E‑commerce

Single‑page cosmetics storefront built from scratch with **HTML**, **CSS**, and **vanilla JavaScript** (DOM only).

## Features
- Hero banner and brand logo (assets included)
- Search, category filter, skin-type filter, price range, sorting
- Product grid with quick view modal
- Cart drawer with localStorage persistence
- Responsive, lightweight and framework‑free

## Getting started
1. Download and unzip the project.
2. Open `index.html` in your browser.
3. Try filters, add to cart, and the demo checkout.

> This is a front‑end demo with mock products. Replace `products` in `app.js` or connect to your API.
